# Glossary

Status: Initial

| Term | Meaning |
|---|---|
| TapTim.e | Product name for the NFC-first time tracking system. |
| ADO | Project-specific organizational knowledge base aligned with FDOS. |
| FDOS | Organizational operating system used as reference for project governance, evidence and learning. |
| Human Architect | Human decision owner for vision, prioritization, business and final approvals. |
| Technical Lead | Responsible role for architecture, implementation quality, maintainability and technical decisions. |
| NFC Tag | Physical NFC chip used to trigger or identify a time tracking action. |
| NFC Scan | Mobile scan action that reads an NFC tag and starts, stops or records a time tracking event. |
| Time Entry | Recorded unit of time tracking data. |
| Evidence | Validated project observation that may inform future decisions or FDOS evolution. |
| ADR | Architecture Decision Record. |

## Glossary Rule

Terms that influence product behavior, architecture or governance must be added here before they become ambiguous.
