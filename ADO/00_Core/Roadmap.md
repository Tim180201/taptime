# Technical Roadmap

Status: Draft

## Sprint 0 – Foundation

Goal: Establish repository, ADO and architecture governance.

Expected outcomes:

- GitHub repository connected
- Project ADO initialized
- ADR process initialized
- Initial risk register created
- Monorepo strategy documented

## Sprint 1 – Architecture Baseline

Goal: Define the initial technical architecture before implementation.

Expected outcomes:

- System overview
- Domain model draft
- Stack decision proposal
- NFC capability model
- Testing strategy draft

## Sprint 2 – Mobile Foundation

Goal: Initialize the mobile application foundation after stack approval.

Expected outcomes:

- Mobile app skeleton
- Navigation baseline
- Configuration strategy
- Local development instructions

## Sprint 3 – Identity and Data Foundation

Goal: Establish authentication, data model and security foundations.

Expected outcomes:

- Auth model
- Data model
- Security rules strategy
- Local/dev/prod environment separation

## Sprint 4 – NFC Engine

Goal: Implement NFC scanning as isolated product capability.

Expected outcomes:

- NFC domain abstraction
- Device validation plan
- Android real-device test evidence
- Failure handling model
